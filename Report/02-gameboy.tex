\section{Der Game Boy und unsere Cartridge}

Mole Mania ist für den Game Boy (Game Boy Classic) entwickelt worden. 
Der Game Boy hat eine Sharp LR35902 CPU verbaut, welches eine Mischung aus 
dem Intel 8080 und dem Zilog Z80 ist. Das heißt, es handelt sich um eine 
8 Bit CPU mit einem 16 bit Adressraum.

Der Speicheraufbau des Game Boy's sieht folgendermaßen aus:
\begin{itemize}
    \item 0000 - 3FFF: ROM Bank00
    \item 4000 - 7FFF: ROM Bank01 - 31
    \item 8000 - 9FFF: VRAM
    \item A000 - BFFF: External RAM
    \item C000 - DFFF: WRAM
    \item E000 - FDFF: Echo RAM
    \item FE00 - FE9F: OAM
    \item FEA0 - FEFF: Not Usable
    \item FF00 - FF7F: I/O Registers
    \item FF80 - FFFE: High RAM (HRAM)
    \item FFFF - FFFF: Interrupt Enable register (IE)
\end{itemize}

Unsere Cartridge hat insgesamt 32 Rombanks, je 16 KiB groß. Das Spiel ist also insgesamt 512 KiB groß.
